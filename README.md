### Running Locally

1. Clone the repository:

    ```bash
    git clone https://github.com/DevVig/youRyse.git
    cd youRyse
    ```

2. Run with Swift Package Manager:

    ```bash
    swift run YouRyse
    ```

    *Or open the folder in Xcode and hit Run.*
